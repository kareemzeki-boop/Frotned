## 2026-04-09 - [Unified Accessibility & Global Keyboard Parity]
**Learning:** In a project with many non-semantic interactive elements (divs as buttons) and multiple overlay systems, implementing a single delegated global `keydown` listener for keyboard parity (Enter/Space to Click) and a centralized `Escape` key handler is significantly more efficient and robust than adding individual listeners to every component. Renaming conflicting global function names (e.g., `closeLightbox`) is also essential to prevent unintended cross-component interactions.
**Action:** Always look for patterns of non-semantic interaction and propose a global delegated listener with `role="button"` checks to solve accessibility at scale. Ensure unique naming for modal/overlay control functions to avoid global scope collisions.
